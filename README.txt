Afin de démarrer le serveur veuillez suivre les étapes suivantes:

Etape 1: Cloner le répertoire Git

Etape 2: Dans le dossier backend, vous trouverez le fichier dot.env, veuillez compléter chacun des 
paramètres par vos identifiants de connexion (nom de la base, nom d'hôte, identifiant et mot de passe)
vous permettant de vous connecter à votre base de donnée MySQL.

Etape 3: Dans la console taper les lignes suivantes:
	- cd frontend
	- cd vue-app
	- npm install
	- npm run serve
Etape 4: Ouvrir un nouvel onglet dans la console et taper:
	-cd backend
	-npm install
	-nodemon server
Dans la console devrait apparaître un message confirmant la connexion à la base de données.
